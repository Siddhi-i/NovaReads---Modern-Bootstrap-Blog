# 🌐 NovaReads - Modern Bootstrap Blog

## 📌 Objective

This project is a modern, responsive blog layout built using Bootstrap 5. It demonstrates the use of UI components, grid system, and JavaScript interactivity for filtering and searching blog posts.

---

## 🚀 Features

* 🔝 Sticky responsive navbar with navigation links
* 🎯 Hero section with call-to-action
* 🧩 Blog cards with images, categories, and descriptions
* 🔍 Search functionality to filter posts by title
* 🗂️ Category filter (Tech, Lifestyle, Education)
* 🎨 Smooth hover animations and transitions
* 📱 Fully responsive design (mobile, tablet, desktop)
* ⚠️ Empty state message when no results found
* 🌐 Footer with social media icons

---

## 🛠️ Technologies Used

* HTML5
* CSS3 (Custom Styling + Animations)
* Bootstrap 5 (CDN)
* JavaScript (DOM Manipulation & Filtering Logic)
* Font Awesome Icons

---

## 🔗 How to Run

1. Clone or download this repository
2. Open `index.html` in your browser
3. Explore blog posts, search, and filter
